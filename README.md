# enfermeria_bakend

Backend del proyecto de gestión de enfermería escolar.

## Tecnologías

- Node.js
- Express.js
- Prisma ORM
- PostgreSQL/MySQL

## Características

- API REST para gestión de pacientes
- Registro de signos vitales
- Autenticación y autorización
- Gestión de usuarios con roles
- Estadísticas y reportes
